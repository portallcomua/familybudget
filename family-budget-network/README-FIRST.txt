Family Budget Network 4.1.0 Beta

Що є у цій збірці:
- PWA з іконками 192/512/maskable/apple-touch/favicon.
- Кнопка встановлення додатка у налаштуваннях.
- Windows .url ярлик через посилання у PWA-блоці.
- Сімейний доступ через запрошення.
- PortmoneDirect / PaymentGateway налаштування та sandbox-ready структура.
- Онлайн-оплата залишається у тестовому режимі до production-доступу/договору.
- Рекламний модуль зі статистикою.
- GitHub auto-update checker через Releases.

Встановлення:
1. Зробити бекап сайту і бази.
2. Деактивувати попередній Family Budget Network.
3. Завантажити цей ZIP через Плагіни -> Додати -> Завантажити.
4. Активувати.
5. Перевірити /my-settings/ -> PWA, сім'ю, Portmone в адмінці.

Для автооновлень:
У GitHub Releases потрібно прикріплювати ZIP-asset з назвою family-budget-network.zip або family-budget-network-*.zip.
